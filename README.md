Book Summary Tool
 About the Project

Book Summary Tool is a simple web application where users can enter a book name and author to get a short summary.

This project is beginner-friendly and helps understand how JavaScript works with user input and conditions.

Features

 Enter book name
 Enter author name
 Get instant summary

Technologies Used

* HTML
* CSS
* JavaScript
